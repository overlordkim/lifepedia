# 人间词条 Lifepedia

**小红书黑客松参赛项目** | 作者：Kashorin & Alvin

> 每个普通人的一生，都值得一整座维基百科。

---

## 这是什么

人间词条是一个 AI 驱动的个人生命百科全书。

你和 AI 对话，讲述你生命中的人、物、地点、事件、时光——AI 把它们编纂成一篇篇维基百科式的词条。不是一篇关于你的百科页面，而是一整座只属于你的维基百科。词条之间用蓝色超链接互相穿越，你的人生变成一张密集的知识图谱。

你外婆的词条里提到了她的红烧肉，点进去是一篇独立的物品词条。红烧肉词条的"相关条目"链接到"春节家庭聚餐"。聚餐词条提到了"长沙岳麓区的老房子"，点进去是一篇地点词条——底部注明"该建筑已于 2021 年拆除"。

---

## 为什么是维基百科

这不是一个随意的形式选择。

日记会写："我好想念外婆做的红烧肉。"

维基百科会写："张薇（1941 年—2019 年），以其独创的红烧肉配方闻名于家族内部。该配方据信源自其母亲口传，未有文字记录。张薇去世后，其外孙女曾多次尝试复原该配方，均未成功。"

第二种更痛。用看待一个文明消亡的冷静去看待你外婆的一道菜——语域的错位本身就是极其有力的情感表达。"均未成功"四个字比"我好想她"杀伤力大十倍。

## 维基百科的原生机制，变成自我反思的工具

我们不只是借用了维基百科的视觉样式，而是把它的核心编辑机制转化成了情感探索的功能：

- **[来源请求]** — 你说"我爸从来不支持我"，AI 生成的词条会给这句话加上 `[来源请求]`。不是反驳你，而是温和地邀请你验证自己反复讲述的叙事。也许你会想起他其实默默帮你交了报名费。
- **消歧义页** — 搜索"那一次我哭了"，系统列出你人生中每一次流泪的独立词条。这个页面本身就是一首诗。
- **编辑历史** — 你 18 岁写的"高中"词条和 28 岁修订的版本共存。你对同一件事的理解如何随时间变化，本身就是一个关于成长的故事。
- **分类系统** — 随着词条积累，AI 自动发现你生命中涌现出的隐藏分类："因为害怕而做的决定""已经消失的地方""在我不相信自己的时候相信了我的人"。
- **中立性争议** — 当你的叙述严重偏向一方，词条顶部会出现"本条目的中立性有争议"横幅，邀请你想象另一个人对同一件事的版本。

## 社交层：你在别人的百科全书里

人间词条最深的社交机制是跨百科链接。

你写了一篇关于大学室友的词条，你室友也在用这个产品，她也写了一篇关于你的词条。点击蓝色链接，你跳转到她的百科全书里——你第一次读到了你在别人生命故事里的样子。

这回答了人类社交中最深的渴望之一：**我在别人眼里是什么人？我在别人的人生里扮演了什么角色？**

## 产品形态

- PWA（Progressive Web App），移动端优先
- AI 对话式创作：和 AI 聊天，词条自动生成并持续丰富
- 七大词条分类：人物、栖居、相伴、滋味、旧物、际遇、流年
- 维基百科风格渲染：信息框、章节、蓝色链接、来源请求、标签
- 分享长图：一键生成精美的词条长图，可保存或分享
- 多人合编：邀请他人一起编纂词条
